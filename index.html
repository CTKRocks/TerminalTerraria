Terminal Terraria

Terminal Terraria is a clone of Terraria written in python for low level machines or computers that can't handle large amounts of graphic display. Please contribute to be recognized. All help is appreciated.

Current Needs

Currently we need ANSI artists or ANSI art generators for converting original Terraria game art to ANSI art. All art will be credited to the creator and RE-Logic.

Project Status

Currently we are working on making Terminal Terraria have the basic land generation. All progress will be charted soon.
